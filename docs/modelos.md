## Modelos del Sistema

---

### Usuario
Representa la cuenta base de todo tipo de usuario en el sistema (participante, jurado o administrador).

**Atributos:**
- `username`: Nombre único de usuario.  
- `email`: Correo electrónico único.  
- `password`: Contraseña cifrada.
